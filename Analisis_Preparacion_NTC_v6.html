Cómo se eligen las tiendas liebre

Una tienda entra solo si cumple las dos condiciones en los últimos 3 meses cerrados:

1
Pedido de 10 ≥ VeteranasSu calidad de pedido (completo × a tiempo × perfecto estado) iguala o supera la media de las tiendas veteranas.
Y
2
Productividad ≤ 0,315 min/ud MOModelo oficial (último − primer evento, menos huecos). Ritmo igual o mejor que el objetivo.

Objetivos para una productividad meta

La base es la productividad global actual (la de la tarjeta de arriba) para el corte elegido. Fijas tu productividad objetivo (en unidades MO o Mercadona) y la herramienta reduce/aumenta cada área y subproceso con el mismo factor (% de mejora del global), proporcional.

Al guardar se descarga Objetivos_Tiendas_Liebre.csv. Súbelo al repo de GitHub (igual que los CSV). El dashboard lo cargará como tienda de referencia «Objetivos Tiendas Liebre» disponible en Comparar y en el Ranking, para medir cualquier tienda contra estos objetivos.
